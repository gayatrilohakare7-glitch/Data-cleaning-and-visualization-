# Data Cleaning and Visualization Project

## Project Overview
This project is a web-based application developed using Python (Flask) for cleaning and visualizing CSV datasets. It allows users to upload a CSV file, automatically clean the data, generate a visualization, preview the cleaned data, and download the cleaned dataset.

---

## Features

- Upload CSV dataset
- Remove duplicate records
- Fill missing values automatically
- Preview cleaned data
- Generate bar chart visualization
- Download cleaned CSV file
- Simple and responsive user interface

## Installation

Install the required Python packages:

```bash
pip install flask pandas matplotlib
```

---

## How to Run

1. Save the backend code as `app.py`.
2. Save the frontend page as `templates/index.html`.
3. Open a terminal in the project folder.
4. Run the following command:

```bash
python app.py
```

5. Open your browser and visit:

```
http://127.0.0.1:5000
```

---

## How to Use

1. Open the application.
2. Upload a CSV dataset.
3. Click the Upload button.
4. The application cleans the data automatically.
5. View the cleaned dataset.
6. View the generated chart.
7. Download the cleaned CSV file.
